## Short Promo Copy (One Paragraph)

Arcade Commander 2.0 is a Windows-based lighting platform built for arcade cabinets. It combines live control, effect authoring, game-specific mapping, and automation in one coordinated suite. With ArcadeCommanderV2, ACLighter, and ACDispatch working together, operators can design, test, and trigger lighting behavior with consistent results across gameplay and frontend events.

## Feature Highlights

- Three coordinated components:
  ArcadeCommanderV2 (operator UI), ACLighter (background lighting service), and ACDispatch (automation CLI).
- Shared effect and animation catalogs:
  Save once and use effects across Commander, Emulator, and FX Editor.
- Fast validation workflow:
  Build and preview in FX Editor, verify in Emulator, then apply live.
- Event-driven automation:
  Trigger animations for game launch, frontend state changes, and custom events.
- Built for operators:
  Includes utility controls like APPLY, ALL OFF, BTN TEST, LED TEST, and DEMO.

## Website Section Copy

### Section: Built for Real Cabinets

Arcade Commander 2.0 is designed for real-world cabinet workflows. Configure controller layouts, map games, and deploy lighting behavior without juggling disconnected tools.

### Section: One Workflow, End to End

1. Configure hardware in Controller Config.
2. Build and preview effects in FX Editor.
3. Assign behavior per game in Game Manager.
4. Validate in Emulator.
5. Run live with ACLighter and automate with ACDispatch.

### Section: Reliable Runtime Architecture

Arcade Commander 2.0 uses centralized runtime data in the `data/` folder for consistent profiles, libraries, and mappings across sessions and updates.
